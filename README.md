# CS649_SkyObjectProject

Libraries required.

Numpy 1.21.5
Pandas 1.4.2
Matplotlib 3.5.1
Seaborn
scikit-learn

Dataset source: http://www.skyserver.org/myskyserver/


How to run code:

Install the above mentioned libraries.
Keep the data folder in the same folder as the Sky_Object.ipynb file
Run the jupyter file.
